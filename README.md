## Project Overview
This repository contains the complete submission for the Digital Literacy project. The work is organized as a five-task portfolio covering digital awareness, online professional presence, coding and collaboration tools, professional communication, and cybercrime awareness.

The repository includes both documentation files and submission evidence such as screenshots, PDFs, and exported outputs. The main editable report is stored in report/project-report.md, and the final exported report is stored in report/Project_Report.pdf.
